Quem sou eu?

Desenvolvedor frontend freelance que transforma visões criativas em sites de alto impacto.
Iniciei minha carreira em 2022, participando de projetos em startups e criando landing pages para projetos independentes. Desde então, me tornei o parceiro confiável de designers que buscam converter protótipos em Figma, Canva, Photoshop e outros em códigos avançados, responsivos, modernos e otimizados para conversão. Desenvolvo páginas simples ou complexas com precisão, além de soluções completas do zero — da UX/UI ao produto final pronto para o ar —, atendendo clientes como escritórios de advocacia, lojas de roupas, escolas de cursos profissionalizantes e muito mais.
Em 2024, decidi investir em minha formação com a graduação em Análise e Desenvolvimento de Sistemas (UniCesumar), reconhecendo que o mercado valoriza bases sólidas.
Minha paixão vai além do código: sou proativo em identificar oportunidades, determinado em superar desafios, e priorizo uma colaboração fluida com comunicação aberta e transparência total. Cada projeto é uma chance de entregar soluções reais que resolvem problemas, impulsionam negócios e geram resultados tangíveis — tornando o processo simples, eficiente e motivador do início ao fim.

Portifólio

Alguns dos meus principais projetos.

Landing Page Curso de Licença Capacitação

Desenvolvi esta landing page completa e otimizada para conversões para promover e impulsionar as vendas de um curso de Licença Capacitação, atendendo a um cliente particular com exigências precisas de design. O projeto construído do zero, incorporando as cores oficiais da logo da empresa para garantir alinhamento visual perfeito, com foco em uma estrutura mobile-first utilizando media queries avançadas para responsividade total em desktops, tablets e smartphones. Implementei animações suaves com CSS transitions e JavaScript para calls-to-action (CTAs) dinâmicos, melhorando a taxa de engajamento. O código é limpo, semântico e otimizado para SEO (com meta tags, alt attributes e estrutura H1-H3), resultando em carregamento rápido (abaixo de 2s) e aprovação imediata pelo cliente após iterações colaborativas transparentes.
Tecnologias utilizadas: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+ para interatividade).

EvandroFlix: Plataforma de Streaming Responsiva e Animada

Aplicação web moderna inspirada em plataformas de streaming, consumindo API de terceiros via Axios para dados em tempo real de filmes e séries. Destaque para a arquitetura componentizada em React com TypeScript, garantindo tipagem estrita, escalabilidade e manutenção facilitada. A responsividade utilizando Tailwind CSS e CSS Grid, adaptando-se perfeitamente a qualquer dispositivo. Animações avançadas implementadas com Framer Motion (via SASS para estilos modulares), incluindo sliders carrossel com hover effects, cards expansíveis e transições fluidas em scroll infinito, elevando a experiência do usuário (UX) a níveis profissionais. O projeto demonstra domínio em gerenciamento de estado (useState/useEffect) e tratamento de erros em chamadas assíncronas, ideal para portfólios de alto impacto.
Tecnologias utilizadas: React, TypeScript, SASS/SCSS, Axios (HTTP client com interceptors).

Landing Page E-commerce: Coleção de Camisas Esportivas

Landing page e-commerce de alto desempenho, projetada para vendas de camisas esportivas, transformando designs criativos em um produto web exclusivo, intuitivo e conversion-oriented. Utilizando uma abordagem progressive enhancement, integrei HTML semântico para acessibilidade (ARIA labels), CSS Grid/Flexbox para layouts adaptáveis e JavaScript para interações dinâmicas como galeria de imagens com zoom, filtros de categoria e animações parallax sutis. Otimizada para performance com lazy loading de imagens e minificação de assets, garante velocidade superior em conexões móveis. O design responsivo cobre todos os breakpoints (mobile, tablet, desktop), com CTAs persuasivos que impulsionam compras, provando minha capacidade de entregar soluções prontas para produção que geram resultados reais.
Tecnologias utilizadas: HTML5, CSS3 (animações keyframe), JavaScript (DOM manipulation e event listeners).

Shopping Cart Interativo: Integração Nativa com Mercado Livre

Carrinho de compras funcional e escalável em React, integrado diretamente à API do Mercado Livre para busca e listagem de produtos em tempo real. Recursos avançados incluem barra de pesquisa com debounce para otimização de API calls, cards de produtos com lazy loading e tooltips, adição/removção de itens via contexto global (React Context API), sidebar persistente com cálculo automático de subtotal/impostos/total, e botão de checkout simulado. A interface é 100% responsiva com hooks customizados (useCallback/useMemo para performance), ícones SVG otimizados (React Icons) e tratamento robusto de estados de loading/error/empty. Este projeto destaca minha expertise em e-commerce frontend, pronto para escalar em produção com persistência local (localStorage).
Tecnologias utilizadas: React (Hooks/Context), Axios/Fetch para API, CSS Modules ou Styled Components.

Flappy Bird: Recreação Clássica com Física Realista

Recriei o icônico Flappy Bird como um jogo web single-page, fiel ao original mas com toques modernos de performance e acessibilidade, durante o curso "Web Moderno + JavaScript" da Udemy. Implementei física realista com requestAnimationFrame para 60 FPS suaves, colisão via Canvas 2D API, sistema de pontuação persistente e restart instantâneo. O canvas é otimizado com image preloading e particle effects sutis em colisões, garantindo jogabilidade fluida em todos os dispositivos. Código modular com OOP (classes para Bird, Pipes, GameLoop) demonstra boas práticas de game dev vanilla, sem frameworks — ideal para entender fundamentos de animação e event handling em JavaScript puro.
Tecnologias utilizadas: HTML5 Canvas, CSS3 (para overlays), JavaScript (ES6 classes e game loop).

Calculadora Interativa: Domínio de Estado e Interatividade em React

Calculadora funcional em React, aplicação prática dos conceitos avançados do curso "Web Moderno + JavaScript" da Udemy, demonstrando gerenciamento de estado reativo e manipulação de eventos do usuário. Arquitetura com hooks nativos como useState para controle dinâmico de valores numéricos, operadores e histórico de operações, com lógica de avaliação de expressões implementada via eval() seguro e parsing customizado para evitar vulnerabilidades (XSS). Incluí tratamento de erros para divisões por zero e entradas inválidas, animações CSS para feedback visual (botões com hover e active states via transitions), e design totalmente responsivo com CSS Grid para layout numérico adaptável em mobile, tablet e desktop. A calculadora suporta operações básicas (soma, subtração, multiplicação, divisão) e avançadas (raiz quadrada, porcentagem), com histórico persistente via localStorage, provando minha capacidade de criar ferramentas interativas escaláveis, ideais para protótipos ou apps educacionais.
Tecnologias utilizadas: React (Hooks: useState, useEffect), CSS3 (Grid/Flexbox para layout responsivo), JavaScript (ES6+ para lógica matemática e event handling).
